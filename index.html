Sync to Google Sheets

🔄 Auto-sync active

One-time setup:

  1. Open Google Sheets → create a new spreadsheet.
  2. Copy the Spreadsheet ID from the URL — it's the long string between /d/ and /edit:
    docs.google.com/spreadsheets/d/15KHWKGQ_qH3sxC…/edit
  3. Click Extensions → Apps Script. Delete all existing code. Paste the script below and Save (Ctrl+S).
  4. Click Deploy → New deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy → authorize when prompted.
  5. Copy the Web App URL (ends in /exec).
  6. Paste both the Spreadsheet ID and Web App URL below → tap Save & enable auto-sync.

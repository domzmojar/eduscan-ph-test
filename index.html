🔐 Managed Sync Setup

If your private Google Sheet was already configured during onboarding, paste your unique Sync URL below to connect this app to your data.

Don't have a Sync URL yet? Set one up yourself →

💻 Do this on a laptop/computer — Apps Script isn't accessible from a phone.

  1. Create a new Google Sheet, then go to Extensions → Apps Script.
  2. Paste in the EduScan PH sync script (ask your administrator or check the EduScan PH documentation for the latest script code).
  3. Click Deploy → New deployment, choose type Web app, set "Who has access" to Anyone, then deploy.
  4. Copy the generated .../exec URL and paste it into the Sync URL field below.

How to install

💡 Once installed, EduScan PH works fully offline — no internet needed to scan. Data syncs to Google Sheets whenever you're back online.

🎥 Installation video

Prefer to watch instead of read? Pick your device below.

📦 Export Data

Export all your data as a JSON backup file — includes classes, students, logs, and settings.

⏱ Auto-Export

Automatically download a JSON backup on a schedule.

⏱ Auto-export active

📥 Import / Restore

Restore data from a previously exported JSON file. You'll see exactly what's in the file before anything is added — nothing is changed until you confirm.

🗑️ Wipe All Data

Permanently erase every class, student, and attendance log from this device. Useful if you imported the wrong backup file and want to start clean. This is only ever reversible from a downloaded backup file — there is no in-app undo.

🗑️ Recently Deleted

Deleted classes, students, and attendance records are kept here for 30 days so you can undo a mistake, even after closing the app. This is separate from sync — nothing here affects your Google Sheets/Supabase data.
